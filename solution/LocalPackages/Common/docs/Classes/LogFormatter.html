  • symbol(for:)

    Provides a special symbol for a given LogLevel.

    The default implementation provides the following symbols:

    • For verbose level: 🔬
    • For debug level: 💬
    • For info level: ℹ️
    • For warning level: ⚠️
    • For error level: ‼️

    For any other LogLevel, the symbol will be 👽.
